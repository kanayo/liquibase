liquibase:diff

Full name:

org.liquibase:liquibase-plugin:1.9.5.0:diff

Description:

A Maven Mojo for performing Database Diffs.

Attributes:

  • Requires a Maven 2.0 project to be executed.
  • Requires dependency resolution of artifacts in scope: test.
